# Project Documentation Guide

This folder contains the design and documentation files for the Demand Forecasting and Inventory Optimization capstone.

- `00_project_brief.md` – Project purpose, scope, and success criteria
- `01_case_study_anchor_target.md` – Real-world case study context
- `02_assumptions_inventory_policy.md` – Modeling and planning assumptions
- `03_planner_playbook_outline.md` – Draft structure for final playbook

These documents were written before implementation and are updated iteratively as the project progresses.
